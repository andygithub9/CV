重要項目和成就

前端工程師 | TVBS公司 | 2022.06 - 現在

  • 項目背景：在 TVBS 面試時，我與主管討論了他們現有網站使用 React 的情況以及網站需要進行服務端渲染（Server Side Rendering, SSR）來提升 SEO 評分和曝光度的重要性。由於 TVBS 的營收主要依賴於曝光和廣告，我提出我熟悉 Next.js，可以幫助解決這個問題。
  • 技術選擇：選擇使用 React 是因為它的高效組件化結構、強大的生態系統和虛擬 DOM 技術，可以大幅提升開發效率和用戶體驗。同時，React 易於集成到現有項目中，並支持各種前端工具和庫。
  • 解決方案：加入團隊後，我將原有的女人我最大網站首頁從 PHP MVC 架構遷移到前後端分離架構的過程。前端採用了 Next.js 框架，實現了服務端渲染（SSR），並結合增量靜態再生（ISR）技術，顯著提升了網站的 SEO 和加載速度。
  • 技術優勢：Next.js 作為 React 的框架，不僅支持服務端渲染（SSR），還提供了增量靜態再生（ISR）功能。這使得我們能夠靈活選擇渲染模式，既保證了熱門頁面的快速加載和高性能，又能應對頻繁更新的內容需求。此外，Next.js 的自動代碼拆分和優化加載路由等內建工具進一步提升了用戶體驗和網站性能。
  • 後續項目：在成功完成女人我最大網站首頁的重構並取得顯著成效後，公司認為我的工作表現優異，隨即指派我負責汽車網站的汽車搜尋頁和汽車規格資訊頁的開發。在這些頁面中，我繼續運用 Next.js 和 React，實現了高效的數據呈現和優化的用戶交互體驗，進一步提升了網站的性能和用戶滿意度。
  • 健康網站團隊：隨後，我被分配到健康網站團隊，負責將 PHP MVC 的前端遷移到 Next.js。此過程中，我面臨了一個重大挑戰：Google 在 2024 年 3 月 12 日將 Interaction to Next Paint (INP) 作為 Core Web Vitals 的指標之一，導致我們網站在 Google 的評分下降。原本評為良好的八萬筆 URL 中，有四萬筆被 Google 判定為需要改善。
  • 挑戰應對與成果：經過多次嘗試和優化，我在五月底成功將四萬筆中三萬七千筆 URL 的 Core Web Vitals 分數提高，並重新提交給 Google 驗證，最終被 Google 認定為良好的 URL。這些優化措施不僅提升了網站性能，也改善了用戶體驗和 SEO 排名。

  • 成果總結：
    • 完成了從 PHP MVC 到前後端分離架構的成功遷移，顯著提升了網站的性能和可維護性。
    • 實現了服務端渲染（SSR）和增量靜態再生（ISR），提高了網站的 SEO 評分和曝光度，為公司帶來了更多的流量和廣告收益。
    • 成功開發了汽車網站的汽車搜尋頁和汽車規格資訊頁，提升了網站的功能性和用戶體驗。
    • 在健康網站團隊中，通過優化措施成功提高了四萬筆中三萬七千筆 URL 的 Core Web Vitals 分數，改善了網站在 Google 的評分。

Key Projects and Achievements at TVBS

Front-end Engineer | TVBS | June 2022 - Present

  • Project Background: During my interview with TVBS, I discussed with the supervisor their current use of React and the need for server-side rendering (SSR) to improve SEO scores and visibility. Since TVBS's revenue heavily relies on exposure and advertisements, I proposed my expertise in Next.js to address this issue.
  • Technology Choice: React was chosen for its efficient component-based architecture, strong ecosystem, and virtual DOM technology, which significantly enhance development efficiency and user experience. Additionally, React is easy to integrate into existing projects and supports various front-end tools and libraries.
  • Solution: After joining the team, I migrate of the homepage of the Woman TVBS website from a PHP MVC architecture to a decoupled front-end and back-end architecture. The front-end was implemented using the Next.js framework, achieving server-side rendering (SSR) and combining incremental static regeneration (ISR) techniques, which significantly improved the website's SEO and loading speed.
  • Technical Advantages: As a React framework, Next.js not only supports server-side rendering (SSR) but also provides incremental static regeneration (ISR) functionality. This allowed us to flexibly choose rendering modes, ensuring fast loading and high performance for popular pages while addressing the need for frequently updated content. Additionally, Next.js’s built-in tools, such as automatic code splitting and optimized route loading, further enhanced user experience and website performance.
  • Subsequent Projects: Following the successful overhaul of the Woman TVBS homepage and achieving significant results, the company recognized my outstanding performance and assigned me to develop the Cars TVBS search page and Cars TVBS specification information page for the automotive website. In these projects, I continued to utilize Next.js and React, achieving efficient data presentation and optimized user interactions, further enhancing website performance and user satisfaction.
  • Health TVBS Website Team: Later, I was assigned to the Health TVBS website team to migrate the front-end from PHP MVC to Next.js. During this process, I faced a major challenge: On March 12, 2024, Google made Interaction to Next Paint (INP) a Core Web Vitals metric, causing our website's scores to drop. Out of the originally well-rated 80,000 URLs, 40,000 were deemed in need of improvement by Google.
  • Challenge Response and Results: Through various attempts and optimizations, by the end of May, I successfully improved the Core Web Vitals scores of 37,000 out of the 40,000 URLs. After resubmitting them to Google for validation, they were recognized as good URLs. These optimization measures not only enhanced website performance but also improved user experience and SEO rankings.
  • Summary of Achievements:
    • Successfully migrated from a PHP MVC to a decoupled front-end and back-end architecture, significantly improving website performance and maintainability.
    • Implemented server-side rendering (SSR) and incremental static regeneration (ISR), increasing the website's SEO scores and visibility, resulting in more traffic and advertisement revenue for the company.
    • Developed the Cars TVBS search page and Cars TVBS specification information page for the automotive website, enhancing the functionality and user experience of the site.
    • In the Health TVBS Website team, successfully improved the Core Web Vitals scores of 37,000 out of 40,000 URLs through optimization measures, improving the website's rating on Google.
